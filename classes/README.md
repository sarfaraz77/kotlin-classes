# Smart Home System – Kotlin OOP Practice

This is a Kotlin learning project created to practice Object-Oriented Programming (OOP) concepts by building a simple Smart Home simulation.

The project models a Smart TV and a Smart Light, both controlled through a central SmartHome class.

# What This Project Covers

# OOP Concepts

Classes & Objects,
Inheritance (IS-A relationship),
Composition (HAS-A relationship),
Encapsulation,
Polymorphism,
Abstraction

# Kotlin Concepts

Visibility Modifiers (private, protected, public),
Property Delegation using ReadWriteProperty,
Custom Getters & Setters,
Constructor Control,
State Management

# Structure

SmartDevice – Base device class

SmartTvDevice – TV with volume & channel control

SmartLightDevice – Light with brightness control

SmartHome – Central controller

RangeRegulator – Handles value limits safely

# Purpose

This project is for learning only, focused on understanding Kotlin OOP fundamentals.
